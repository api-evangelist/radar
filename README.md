Use Radar APIs as building blocks for location-based products and services like pickup and delivery tracking, location-triggered notifications, location verification, store locators, address autocomplete, and more. Or, use Radar APIs to manage your Radar data, including users, geofences, and events.

## APIs

### Radar

Use Radar APIs as building blocks for location-based products and services. Manage users, geofences, and events; perform geocoding, search, and routing.

- **Documentation:** https://radar.com/documentation/api
- **Base URL:** `https://api.radar.io/v1`
- **OpenAPI:** [openapi/radar-openapi.yml](openapi/radar-openapi.yml)
- **Authentication:** API key via `Authorization` header (publishable or secret keys)

#### Resource Categories

- **Track** — User location updates and event generation
- **Geocoding** — Forward, reverse, and IP geocoding
- **Search** — Autocomplete, user/geofence/place search, address validation
- **Routing** — Distance, matrix, match, directions, optimization
- **Users** — CRUD on user records
- **Geofences** — Manage geographic boundaries
- **Events** — Query location events
